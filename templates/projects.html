Mortgage Masters


A team project that trained a machine learning algorithm to determine if a user would be approved for a mortgage loan. We tested several different algorithms and ultimately chose to train a Neural Network using TensorFlow for our application. The flask app takes 10 different inputs from the user and immediately provides a decision.
